Talk to your Copilot. Out loud.

Vox is a Copilot CLI extension that opens a hands-free voice panel for any session. Speak your turn, watch the orb listen, and hear the agent reply. Voice in, voice out — Windows, macOS, and Linux.

Install now View on GitHub

Pure JS. One slash-command. Sibling to Engram & Anya.

Windows · PowerShell
> irm https://raw.githubusercontent.com/aasis21/vox/main/install.ps1 | iex
macOS / Linux · bash
$ curl -fsSL https://raw.githubusercontent.com/aasis21/vox/main/install.sh | bash
Then, in any session
> /vox — open localhost:4321 and tap the orb
01 · What it does

A voice canvas for any session.

Run /vox and a single fixed UI comes up on port 4321 with a reactive listening orb. Spoken turns route to the active session; replies are read back. Many sessions can be live at once.

  1. 01

    Voice in.

    Talk to the agent out loud — your microphone feeds straight into the active Copilot session.

  2. 02

    Voice out.

    Replies are synthesized and read back, so you can keep your hands and eyes on the work.

  3. 03

    One UI, many sessions.

    /vox-who lists every live session; only the active one hears you. Switch any time.

  4. 04

    Cross-platform.

    Pure JavaScript, no build. One-line install on Windows, macOS, and Linux.

02 · Install

One line. That's it.

Requires Node.js and git on PATH. The bootstrap clones the repo to ~/vox and copies the extension into ~/.copilot/extensions/vox, where Copilot CLI auto-discovers it.

# Windows (PowerShell)
irm https://raw.githubusercontent.com/aasis21/vox/main/install.ps1 | iex

# macOS / Linux (bash)
curl -fsSL https://raw.githubusercontent.com/aasis21/vox/main/install.sh | bash

# Uninstall
.\uninstall.ps1      # Windows
./uninstall.sh       # macOS / Linux
03 · Commands

Just type /vox.

CommandPurpose
/voxStart voice mode and make this session the active voice target. Opens localhost:4321.
/vox-stopStop Vox for this session and release its voice server.
/vox-whoList live Vox sessions and show which one is active.
